## 🚀 Running Locally

To run this project on your local machine:

1. Clone the repository:
   ```bash
   git clone <your-github-repo-url>
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Start the development server:
   ```bash
   npm run dev
   ```
